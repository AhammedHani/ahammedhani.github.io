Linux Commands Cheat Sheet

August 13, 2024

2024

Welcome to the ultimate Linux commands cheats! Whether you’re a beginner or a seasoned developer, knowing the basic Linux commands can significantly boost your productivity. Here’s a comprehensive list of essential commands categorized for ease of use:

Basic Commands

  • List Files and Directories
    • ls -la - List all files and folders in the current directory, including hidden ones.
    • ls -R - Recursively list files in all sub-directories.
  • Directory Navigation
    • pwd - Print the current working directory.
    • cd - Go to the home directory.
    • cd .. - Move up one directory level.
    • cd - - Switch to the previous directory.
  • Command History
    • history - Display command history.
    • history | grep command - Search command history for a specific command.
  • User Information
    • whoami - Show the currently logged-in username.
  • Manual and Help
    • man command - Display the manual for a specific command.
    • tldr command - Provide simplified help with examples for a command.

File Operations

  • Viewing and Editing Files
    • cat filename.txt - Display the content of a file.
    • cat > filename1.txt - Create a new file and write to it.
    • touch filename2.txt - Create an empty file or update the timestamp of an existing file.
    • less filename.txt - View the file content page by page.
    • head filename.txt - View the first 10 lines of a file.
    • tail filename.txt - View the last 10 lines of a file.
  • File Management
    • cp sourcefile.txt /destination/path/ - Copy a file to a new location.
    • mv oldname.txt newname.txt - Rename a file.
    • mv filename.txt /new/path/ - Move a file to a new location.
    • file filename.txt - Determine the file type.
    • rm filename.txt - Delete a file.
    • rmdir directoryname - Delete an empty directory.

Directory Operations

  • Directory Management
    • mkdir directoryname - Create a new directory.
    • cp -r /path/to/source /path/to/destination - Recursively copy a directory and its contents.
    • rm -rf directoryname - Recursively delete a directory and its contents.
    • tree - Display directories in a tree-like format.

Permissions

  • Setting Permissions
    • chmod 755 filename.txt - Change file permissions to read, write, and execute for the owner; read and execute for the group and others.
    • chown username:groupname filename.txt - Change the owner and group of a file.

System Monitoring

  • Resource Monitoring
    • top - Display ongoing processes and their resource usage.
    • htop - An enhanced version of top with a better user interface.
  • Memory and Swap Usage
    • free -h - Display total, used, and free memory and swap usage in a human-readable format.

Disk Usage

  • Space Usage
    • du -sh /path/to/directory - Report the size of a directory and its contents.
    • df -h - Display available disk space in a human-readable format.

Networking

  • Network Connections
    • ping -c 4 google.com - Send four packets to check the connectivity to Google.
    • ifconfig - Display or configure network interfaces.
    • netstat -tulnp - Display active listening ports and their associated programs.
  • SSH
    • ssh user@hostname - Connect to a remote server via SSH.
    • scp file.txt user@hostname:/path/to/destination - Securely copy a file to a remote server.

Process Management

  • Process Control
    • ps aux - List all running processes with detailed information.
    • kill PID - Terminate a process by its Process ID.
    • killall processname - Terminate all processes with the specified name.

User Management

  • User Accounts
    • adduser username - Add a new user to the system.
    • deluser username - Remove a user from the system.
    • passwd username - Change the password for a user.

Package Management

  • Package Installation and Removal
    • apt-get update - Update the list of available packages.
    • apt-get install package - Install a new package.
    • apt-get remove package - Remove an installed package.
    • apt-get upgrade - Upgrade all installed packages to their latest versions.

Search and Filtering

  • Searching and Filtering
    • grep 'search_term' filename.txt - Search for a term within a file.
    • find /path -name 'filename' - Locate files with a specific name within a path.
    • locate filename - Find files by name, quickly searching a database of file paths.

System Information

  • System Overview
    • uname -a - Display detailed information about the system.
    • lsb_release -a - Show the distribution details.
    • lscpu - Display CPU architecture information.
